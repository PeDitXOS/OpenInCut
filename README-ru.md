# OpenInCut

Кроссплатформенный видеоредактор для создателей контента с суперсилами ИИ: **текстовый монтаж** (Whisper по словам), **удаление тишины одним кликом**, **автоматические вертикальные видео**, **реактивный аватар на эмоции**, **субтитры в стиле караоке**, **озвучка из текста** и **встроенный MCP-сервер**, чтобы ИИ-агент мог редактировать ваш проект.

## ✨ Возможности

### Монтаж по тексту
Расшифровывайте видео с Whisper (таймкоды по словам), затем редактируйте, удаляя слова в транскрипции — видео обрезается автоматически.

### Монтаж на базе ИИ
Встроенный MCP-сервер с 53 инструментами. Подключите Claude, ChatGPT или любой LLM для редактирования проекта на естественном языке:
- «Добавь заголовок: Добро пожаловать»
- «Убери тишину с 0:15 до 0:20»
- «Исправь перспективу в этом клипе»
- «Добавь русские субтитры»

### Цветокоррекция и эффекты
- **Цветокоррекция**: яркость, контраст, насыщенность, гамма
- **Chroma Key**: удаление зелёного/синего фона
- **Гауссово размытие**: размытие фона
- **Drop Shadow**: тени для наложений
- **Исправление перспективы**: выравнивание горизонта (с помощью ИИ)
- **Вертикальный фон**: размытый фон для вертикального видео

### Субтитры и TTS
- Автосубтитры из транскрипции (по словам или в режиме караоке)
- Поддержка русского языка
- Направление текста RTL для языков справа налево
- Text-to-Speech с Kokoro-82M (54 голоса, 8 языков)

### Монтаж таймлайна
- Drag-and-drop таймлайн с магнитным прилипанием
- Мультитрек композитинг
- Ключевые кадры на всех свойствах
- 11 типов переходов (crossfade, wipe, slide, dissolve и др.)
- Контроль скорости (0.25x–4x) с сохранением тона

### Экспорт
- Форматы MP4, WebM, GIF
- Настраиваемые разрешение и качество
- Пакетный экспорт с диапазонами

## 🔌 MCP-сервер

OpenInCut запускает MCP-сервер на `http://127.0.0.1:4599/mcp` при запуске. Любой ИИ-агент, понимающий MCP, может редактировать ваш проект:

```bash
claude mcp add --transport http opencut http://127.0.0.1:4599/mcp \
  --header "Authorization: Bearer YOUR_TOKEN"
```

53 инструмента доступны — см. [docs/MCP.md](docs/MCP.md).
