# OpenInCut

محرر فيديو متعدد المنصات مصمم لمنشئي المحتوى، مع قدرات ذكاء اصطناعي خارقة: **تحرير نصي** (Whisper كلمة بكلمة)، **إزالة الصمت بنقرة واحدة**، **أفقيات تلقائية**، **أفاتار تفاعلي مع المشاعر**، **ترجمة كاريوكي**، **تعليق صوتي من النص**، و**خادم MCP مدمج** لكي يتمكن وكيل الذكاء الاصطناعي من تحرير مشروعك.

## ✨ الميزات

### التحرير النصي
انسخ فيديو الخاص بك مع Whisper (طوابع زمنية على مستوى الكلمة)، ثم حرر عن طريق حذف الكلمات في النص — يتم قطع الفيديو تلقائياً.

### تحرير مدعوم بالذكاء الاصطناعي
خادم MCP مدمج بـ 53 أداة. وصل Claude، ChatGPT، أو أي LLM لتعديل مشروعك بلغة طبيعية:
- "أضف عنواناً يقول أهلاً"
- "أزل الصمت من 0:15 إلى 0:20"
- "صحح المنظور في هذا المقطع"
- "أضف ترجمة عربية"

### تصحيح الألوان والمؤثرات
- **تصحيح الألوان**: السطوع، التباين، التشبع، الجاما
- **Chroma Key**: إزالة الشاشة الخضراء/الزرقاء
- **Gaussian Blur**: تمويه الخلفية
- **Drop Shadow**: تأثيرات الظل للتراكبات
- **تصحيح المنظور**: تسوية الأفق (مُساعد بالذكاء الاصطناعي)
- **Vertical Fill**: خلفية مموهة للفيديوهات العمودية

### الترجمة والتعليق الصوتي
- ترجمة تلقائية من النسخ (كل كلمة أو وضع الكاريوكي)
- دعم اللغتين الفارسية والعربية
- اتجاه نص RTL للغات المكتوبة من اليمين لليسار
- تحويل النص للكلام مع Kokoro-82M (54 صوت، 8 لغات)

### تحرير خط الزمن
- خط زمني بالسحب والإفلات مع التثبيت المغناطيسي
- تراكب متعدد المسارات
- رسوم متحركة بإطارات مفتاحية على جميع الخصائص
- 11 نوع انتقال (تلاشي متقاطع، مسح، شريحة، حل، إلخ)
- تحكم في السرعة (0.25x إلى 4x) مع الحفاظ على النغمة

### التصدير
- صيغ MP4، WebM، GIF
- دقة وجودة قابلة للتخصيص
- تصدير دفعي مع نطاقات القطع

## 🚀 البدء السريع

### المتطلبات
- FFmpeg 6 أو أحدث في PATH
- Rust (stable) + Node 20 أو أحدث

### التثبيت
```bash
git clone https://github.com/PeDitXOS/OpenInCut.git
cd OpenInCut
npm install
npx tauri dev
```

أو حمل النسخة الجاهزة من [GitHub Releases](https://github.com/PeDitXOS/OpenInCut/releases).

## 🔌 خادم MCP

يعمل OpenInCut بخادم MCP على `http://127.0.0.1:4599/mcp` عند البدء. أي وكيل ذكاء اصطناعي يتحدث MCP يمكنه تحرير مشروعك:

```bash
claude mcp add --transport http opencut http://127.0.0.1:4599/mcp \
  --header "Authorization: Bearer YOUR_TOKEN"
```

53 أداة متاحة — راجع [docs/MCP.md](docs/MCP.md).
