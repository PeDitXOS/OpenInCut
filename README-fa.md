# OpenInCut

ویرایشگر ویدیویی چندپلتفرمی برای سازندگان محتوا، با قدرت هوش مصنوعی: **ویرایش متنی** (Whisper کلمه به کلمه)، **حذف سکوت با یک کلیک**، **عمودی‌سازی خودکار**، **اواتار با واکنش احساسی**، **زیرنویس کارائوکه**، **صوت از متن**، و **سرور MCP تعبیه شده** تا عامل هوشمند بتواند پروژه‌تان را ویرایش کند.

## ✨ ویژگی‌ها

### ویرایش متنی
رونویسی ویدیو با Whisper (برچسب‌های زمانی سطح کلمه)، سپس ویرایش با حذف کلمات در متن — ویدیو به‌صورت خودکار برش می‌خورد.

### ویرایش هوشمند با هوش مصنوعی
سرور MCP داخلی با ۵۳ ابزار. وصل کردن Claude، ChatGPT، یا هر LLM برای ویرایش پروژه با زبان طبیعی:
- «یک عنوان بگذار بگه خوش‌آمدید»
- «سکوت بین ۰:۱۵ تا ۰:۲۰ رو حذف کن»
- «منظور این کلیپ رو تصحیح کن»
- «زیرنویس فارسی اضافه کن»

### تصحيح رنگ و افکت‌ها
- **تصحیح رنگ**: روشنایی، تضاد، اشباع، گاما
- **Chroma Key**: حذف سبز/آبی‌پوش
- **Gaussian Blur**: تار کردن پس‌زمینه
- **Drop Shadow**: سایه برای لایه‌ها
- **تصحیح پرسپکتیو**: تراز افق (با کمک هوش مصنوعی)
- **Vertical Fill**: پس‌زمینه تار برای ویدیو عمودی

### زیرنویس و TTS
- زیرنویس خودکار از رونویسی (به‌صورت کلمه به کلمه یا کارائوکه)
- پشتیبانی از زبان‌های فارسی و عربی
- جهت متن RTL برای زبان‌های راست‌به‌چپ
- تبدیل متن به صدا با Kokoro-82M (۵۴ صدا، ۸ زبان)

### خط زمان ویرایش
- خط زمان کشیدن-انداختن با چسبیدگی مغناطیسی
- ترکیب چندمساره
- انیمیشن کلید در کلید روی همه ویژگی‌ها
- ۱۱ نوع انتقال (تلاش متقاطع، ماسح، لغزش، حل، و غیره)
- کنترل سرعت (۰.۲۵x تا ۴x) با حفظ نغمه

### صادرات
- فرمت‌های MP4، WebM، GIF
- رزولوشن و کیفیت قابل تنظیم
- صادرات دسته‌ای با محدوده‌های برش

## 🚀 شروع سریع

### پیش‌نیازها
- FFmpeg ۶ یا جدیدتر در PATH
- Rust (stable) + Node ۲۰ یا جدیدتر

### نصب
```bash
git clone https://github.com/PeDitXOS/OpenInCut.git
cd OpenInCut
npm install
npx tauri dev
```

یا نسخه از پیش ساخته را از [GitHub Releases](https://github.com/PeDitXOS/OpenInCut/releases) دانلود کنید.

## 🔌 سرور MCP

OpenInCut سرور MCP را روی `http://127.0.0.1:4599/mcp` در هنگام راه‌اندازی اجرا می‌کند. هر عامل هوش مصنوعی که MCP می‌فهمد می‌تواند پروژه‌تان را ویرایش کند:

```bash
claude mcp add --transport http opencut http://127.0.0.1:4599/mcp \
  --header "Authorization: Bearer YOUR_TOKEN"
```

۵۳ ابزار موجود — [docs/MCP.md](docs/MCP.md) را ببینید.

## 📜 اعتبار

OpenInCut بر پایه [UberEditor](https://github.com/HectorPulido/UberEditor) از **HectorPulido** ساخته شده. از کار اصلی روی معماری سرور MCP، موتور خط زمان، و چارچوب یکپارچه‌سازی هوش مصنوعی ممنونیم.

**مجوز**: Apache-2.0 (همانند اصلی). کپی‌رایت ۲۰۲۶ PeDitXOS و مشارکت‌کنندگان.
